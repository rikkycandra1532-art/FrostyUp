Kebijakan Privasi

Terakhir diperbarui: 1 Juli 2026

FrostyUp ("kami", "layanan kami") berkomitmen untuk melindungi privasi setiap pengguna ("kamu") yang menggunakan platform kami di frostyup.vercel.app. Kebijakan privasi ini menjelaskan jenis data apa yang kami kumpulkan, bagaimana kami menggunakannya, dan hak-hak yang kamu miliki atas data tersebut.

1. Data yang Kami Kumpulkan

Dalam proses penggunaan layanan, kami dapat mengumpulkan data berikut:

  • Data transaksi: User ID game, Zone ID, nama produk yang dibeli, jumlah transaksi, dan kode invoice.
  • Data kontak: Nomor WhatsApp dan alamat email yang kamu masukkan secara sukarela untuk pengiriman invoice dan konfirmasi transaksi.
  • Data pembayaran: Metode pembayaran yang dipilih dan status transaksi. Kami tidak menyimpan nomor kartu kredit, nomor rekening, atau data sensitif pembayaran lainnya.
  • Data teknis: Alamat IP, jenis browser, dan halaman yang dikunjungi — dikumpulkan secara anonim untuk keperluan keamanan dan peningkatan layanan.

2. Penggunaan Data

Data yang kami kumpulkan digunakan untuk:

  • Memproses dan menyelesaikan transaksi top up game yang kamu lakukan.
  • Mengirimkan invoice, konfirmasi pesanan, dan notifikasi status transaksi.
  • Memberikan dukungan pelanggan jika terjadi masalah pada transaksi.
  • Meningkatkan keamanan sistem dan mencegah penipuan atau penyalahgunaan layanan.
  • Meningkatkan kualitas layanan berdasarkan pola penggunaan secara agregat dan anonim.

3. Penyimpanan Data

Data transaksi disimpan di server aman berbasis Supabase (PostgreSQL) dengan enkripsi data dalam penyimpanan. Kami menyimpan data transaksi selama minimal 12 bulan untuk keperluan rekonsiliasi dan dukungan pelanggan, setelah itu data dapat dihapus atau dianonimkan.

4. Berbagi Data dengan Pihak Ketiga

Kami tidak menjual, menyewakan, atau membagikan data pribadi kamu kepada pihak ketiga untuk keperluan pemasaran. Kami hanya berbagi data dengan pihak ketiga berikut untuk keperluan operasional layanan:

  • Midtrans (payment gateway): Untuk memproses pembayaran secara aman. Data yang diteruskan: nominal transaksi dan detail produk. Diatur oleh kebijakan privasi Midtrans.
  • Digiflazz (penyedia produk digital): Untuk memproses pengiriman item top up ke akun game. Data yang diteruskan: User ID game dan produk yang dibeli.
  • Supabase (database): Penyedia infrastruktur database tempat data transaksi disimpan secara aman.

5. Keamanan Data

Kami menerapkan langkah-langkah teknis dan organisasi yang wajar untuk melindungi data kamu, termasuk enkripsi data dalam transit (HTTPS/TLS), Row Level Security di database, dan pembatasan akses ke data oleh staf yang memerlukan.

Namun, tidak ada sistem yang 100% aman. Jika terjadi kebocoran data yang memengaruhi kamu, kami akan memberitahu kamu sesegera mungkin.

6. Hak-hak Pengguna

Kamu memiliki hak untuk:

  • Meminta akses ke data pribadi yang kami simpan tentang kamu.
  • Meminta koreksi data yang tidak akurat.
  • Meminta penghapusan data transaksi (dengan catatan, beberapa data mungkin perlu dipertahankan untuk keperluan hukum).
  • Mengajukan keberatan atas penggunaan data kamu.

Untuk menggunakan hak-hak ini, hubungi kami di support@frostyup.id.

7. Cookie

Website kami menggunakan cookie minimal yang diperlukan untuk fungsi dasar situs (seperti preferensi tema). Kami tidak menggunakan cookie pelacakan pihak ketiga untuk keperluan iklan.

8. Perubahan Kebijakan

Kami dapat memperbarui kebijakan privasi ini sewaktu-waktu. Perubahan akan diumumkan di halaman ini dengan memperbarui tanggal "Terakhir diperbarui". Penggunaan layanan kami setelah perubahan dianggap sebagai persetujuan terhadap kebijakan yang diperbarui.

9. Hubungi Kami

Jika ada pertanyaan mengenai kebijakan privasi ini, hubungi kami di:

  • Email: support@frostyup.id
  • WhatsApp: via tombol live chat di website
